About me

안녕하세요, Front-End 개발자 이연성입니다. :D

React와 TypeScript 기반의 웹/앱 개발을 중심으로, 다양한 분야의 솔루션을 기획하고 구현해 왔습니다. 사용자 경험을 최우선으로 고려하며, 다음과 같은 기술 과제를 해결해 왔습니다.

  • 음성 인식 기반 시스템: 산업 현장 및 병원 데이터를 음성으로 수집하고 정제하는 시스템을 구현하여, 머신러닝 학습에 최적화된 데이터 파이프라인을 구성했습니다.
  • WebRTC 기반 실시간 영상 연결: 비대면 진료 및 원격 회의 시스템에서, 1:1 영상통신부터 다자간 Agora SDK 연동까지 확장하며 안정적인 커뮤니케이션 환경을 구축했습니다.
  • 대용량 데이터 처리 UI: 최대 10만 건 이상의 데이터를 렌더링하는 플랫폼에서 가상화 리스트, 무한 스크롤, skeleton UI 등을 적용하여 사용성을 극대화하고 렌더링 속도를 80% 이상 개선했습니다.
  • 상태 관리 최적화: Redux, Recoil, Zustand, React Query 등 다양한 상태 관리 도구를 사용해 복잡한 상태 흐름을 명확히 정리하고, 클라이언트/서버 상태를 효과적으로 분리해 관리했습니다.

기술적인 문제 해결뿐 아니라, 기획자와의 협업, 디자인 없이 UI/UX를 설계해 구현하는 능력, 테스트 기반의 안정성 확보에도 관심이 많습니다.

무엇보다도, 동료와 함께 성장하는 문화를 만들고 유지하는 개발자가 되고자 노력하고 있습니다.
